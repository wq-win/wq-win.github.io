如何搭建 GitHub.io 个人博客

2023年10月13日

GitHub Pages 是 GitHub 提供的静态网站托管服务，适合搭建个人博客、项目文档等。以下是详细步骤：

步骤1：创建 GitHub 仓库

登录 GitHub 后，创建一个名为 username.github.io 的仓库（将 username 替换为你的 GitHub 用户名），仓库必须设置为公开（Public）。

步骤2：编写静态页面

使用 HTML、CSS、JavaScript 编写博客页面，首页建议命名为 index.html，这是 GitHub Pages 的默认入口文件。

步骤3：部署网站

将代码推送到仓库的 main 分支，GitHub 会自动部署网站。大约 1-2 分钟后，访问 https://username.github.io 即可看到你的博客。

进阶技巧：

  • 可以使用 Jekyll 等静态网站生成器自动生成博客框架
  • 通过自定义域名绑定，将博客绑定到自己的域名（如 example.com）
  • 使用 GitHub Actions 实现自动部署和更新
